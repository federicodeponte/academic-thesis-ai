# OpenDraft Website

Official landing page for OpenDraft - built with Next.js 14.

## Quick Start

```bash
cd website
npm install
npm run dev
```

Visit http://localhost:3000

## Features

- Viral waitlist system with 5-tier gamified referrals
- Dual-path hero CTA (Waitlist + GitHub)
- Anti-fraud protection
- Real-time stats
- Email verification
- Responsive design with shadcn/ui

## Deploy

```bash
npm run build
vercel --prod
```
